Offerings

We provide APIs to detect/ recognize objects from images/ videos to empower self driving cars using our state of the art algorithms based upon Computer Vision and Deep Learning.



Traffic Signs Recognition

Traffic Signs recognition API detects more than 100 classes of Indian traffic signages like Speed Limit, Traffic Signal, Stop Signs etc.



Lane Detection

Lane Detection API detects lane markings on Indian roads



Image Segmentation

Image Segmentation API segments the image in various parts like road, humans, vehicles, buildings, sky etc
